# Diatotab
Diatotab is a Javascript application used to generate tablature for diatonic accordion / melodeon and harmonica.

The application can be used [here](https://diatotab.com), directly from the webbrowser.

Tablature generation is based on a fork of abcjs which is maintained in a separate [repository](https://github.com/RickvanderS/abcjs-melodeon-tab).

The abc editor is based on CodeMirror, the compiled editor is found in this [repository](https://github.com/RickvanderS/codemirror-editor-abcjs). The 'lezer' to parse the ABC grammer is in this [repository](https://github.com/RickvanderS/codemirror-lang-abcjs).
